Key features of Sass

(Syntactically awesome style sheets)

Go to the original website: https://www.netguru.com/blog/7-sass-features-you-should-be-familiar-with

1. Reference symbol ( & )

.block {
  &.red {
      color: red;
  }
}

2. Partials and @import directive

Bu yöntem, diğer kod bitlerini soyutlamaya ve ayrı tutmaya yardımcı olur (karışımlar, ayarlar (tanımlı değişkenler), vb. için). (Long story.)

/* posts.scss */.posts{
  &.show{
    div.posts_list{
      div.post{
        background: whitesmoke;
        p.post_title{
          font-size: 16px;
        }
      }
    }
  }
}

3. Interpolation

Bir değişkende bir eleman tanımlayabilir ve onu sass kodumuza enterpolasyon yapabiliriz. Bu, modüllerinizi ayrı dosyalarda tuttuğunuzda kullanışlıdır:

$el : "post";
.#{$el} {
    background: blue;
    & .#{$el}__title {
        background: red;
    }
}

4. @content directive

Since SASS 3.2.0 we can pass a code block into a mixin:

@mixin you-can-do-it {
    @content;
}

@include you-can-do-it {
  .yes-you-can { color: red }
}

5. %placeholders

SASS 3.2.0'dan beri kullanılabilen yer tutucular, genişletilmesi gereken stiller yazmak istediğinizde, ancak temel stillerin çıktı css stillerinde (@expanding sınıfların aksine) görünmesini istemiyorsanız kullanışlıdır.

%post-skeletor{
    font:{ size: 12; weight: 700;}
    box-sizing: border-box;
    padding: 5px;
    margin: 10px;
    border: 1px solid whitesmoke;
}

.post-class {
    font:{ size: 12; weight: 700;}
    box-sizing: border-box;
    padding: 5px;
    margin: 10px;
    border: 1px solid whitesmoke;
}

.post1--extends__placeholder {
    @extend %post-skeletor;
    background: yellow;
}

.post1--extends_class{
    @extend .post-class;
    background: yellow;
}

6. Functions

Sass, bazıları renkle, bazıları dizelerle ve bazıları da sayılarla ilgili bir dizi işleve sahiptir.

.post--big{
  width: percentage(.5);
  font-size: max(12px, 14px, 16px);
  background: rgba(231, 0.8, 2, .8);
}

$title: 'This is title';
.post__title{
  content: quote($title);
}

You can also define your own functions like so:

@function double($variable){
 @return $variable *2;
}

.post{
  font-size: double(2px);
}

7. Lists and @each directive

Listeler, Sass'ın eski bir özelliğidir, tek başlarına pek bir şey yapamazlar. Ancak liste işlevleri (join, lenght, nth, append) ve @each yönergesi ile birleştirildiğinde işler daha ilginç hale gelir, liste kullanımı için mükemmel bir örnek hareketli simgelerdir (sprite icons).

$icons: smile, poop, godzilla;

@each $icon in $icons {
  .icon-#{$icon} { background-image: url('/icons/#{$icon}.png') }
}
